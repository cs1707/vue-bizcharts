# vue-bizcharts

基于 G2 封装 React 图表库，可以让用户以组件的形式组合出无数种图表 

BizCharts 的 Vue 版本 （非官方）


## Install
```
npm i vue-bizcharts -S
```

## Quick Start
``` javascript
import Vue from 'vue'
import BizCharts from 'vue-bizcharts'

Vue.use(BizCharts)
```

## Warning
目前还是 alpha 版本，若在生产环境中使用请慎重
